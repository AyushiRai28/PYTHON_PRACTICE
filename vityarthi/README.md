# Student–Faculty Management & Examination Info System

This project is a Python-based interactive console application designed for *Dumplin University*.  
It collects and manages *student/faculty information, displays **subject details, and provides **examination schedules* based on user input.

---

## 📌 Features

### *1. User Management*
- Supports two user types: *Student* and *Faculty*
- Validates basic personal information
- Displays user-specific details in a structured format

### *2. Stream & Subject Display (for Students)*
Streams supported:
- PCM
- PCB
- Arts
- Commerce  
The system automatically prints all subjects associated with the chosen stream.

### *3. Examination Schedule*
- Displays exam dates for:
  - PCM
  - PCB
  - Arts
  - Commerce
- Shows eligibility criteria (attendance, midterm marks, previous grades)

### *4. Basic Error Handling*
- Age validation  
- Phone number validation  
- Invalid user-type handling  
- Invalid stream handling
